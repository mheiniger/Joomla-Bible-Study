Teachers Help Screen

Click on the New link to add a new teacher. Click a link to edit a teacher, or the checkbox followed by Edit on the top menu.

Put in a name for the teacher and click Save. You can edit them. To delete rows you need to set the Allow Admin Delete Rows option under Preferences. But be careful. If you delete a teacher that is attached to a study it may cause unintended results.
